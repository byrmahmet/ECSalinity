Calibrates the connected probe and saves the result in EEPROM numberProbeSamples measurements are made, the median third is taken and averaged together to get a K value for the probe.

Parameters
solutionECthe EC of the calibration solution in mS
tempCoefthe coefficient used to calibrate the probe
numberProbeSamplesthe number of samples to take to calibrate the probe (must be a multiple of 3)
Postcondition
K will be saved in EEPROM and used automatically thereafter

float EC_Salinity::measureEC ( float  tempCoefficient)

Starts an EC measurement.

accuracy of samples are taken and the average of the middle third of the measures is returned in mS. If useTempCompensation is TRUE, then the resulting measurement will be corrected for temperature. If tempCompensationConstant is set to anything other than -1, that value will be used as the temperature.

Parameters
tempCoefficientthe coefficient used to compensate for temperature.
Precondition
K must be set
Postcondition
uS, mS, S, tempC, tempF, PPM_500, PPM_640, PPM_700, salinityPPM, salinityPPT, salinityPSU, and K are updated
Note
PPM is not valid if salinity is being measured
Returns
mS
