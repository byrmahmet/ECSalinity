void EC_Salinity::setAccuracy ( byte  b)

Configures the accuracy of the device.

The device maintains a running median of values. It throws out the top and bottom third of values, then averages the middle third together to return a single value. The accuracy incrases with a high number. It must be evenly divisible by 3.

Parameters
baccuracy of the device

void EC_Salinity::setTempConstant ( byte  b)

Configures device to use the provided temperature constant.

By default, the temperature constant is set to -1 which instructs the actual temperature to be used for temperature compensation, however any number can be specified. To use the actual temperature, restore the value to -1.

Parameters
bthe temperature to use for compensation
